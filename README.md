# TypeScript Basics

This repository contains basic TypeScript programs for learning the fundamentals of TypeScript.

## Topics Covered
- Type Annotations
- Data Types (Number, String, Boolean)
- Functions
- Normal Parameters
- Default Parameters
- Optional Parameters
- Rest Parameters
- Return Types
- Arrow Functions
- Function to Arrow Function Conversion
- Classes and Objects
- Constructors
- Access Modifiers
- Readonly Members
- Static Members

## Technologies Used
- TypeScript
- Node.js
- Visual Studio Code
